# TP-Link Tapo Smart Plug

This module controls TP-Link Tapo Smart Plugs and Bulbs.

## Setup

Ensure that Third-Party Compatibility is [enabled](https://www.tapo.com/uk/faq/714/).
Once enabled you may need to restart the device.

To enable go to `Tapo App > Me > Third-Party Services > Third-Party Compatibility > On`.

## Actions:

* Power On/Off/Toggle
* Set Bulb Brightness
* Set Bulb Color

## Feedbacks:

* Power On/Off

## Variables:

* Device ID
* FW, HW Version
* Model
* Mac Address
* HW, FW, OEM ID
* On Time
* Overheated State
* Nickname
* Location
* Latitude/Longtitude
* SSID/Signal Level/RSSI
* Power State

## Presets

* Power On/Off
